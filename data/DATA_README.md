# Data package

This folder provides shareable derived datasets needed to interpret, verify and extend the reported analyses. It includes model tables, grid outputs, selected derived geospatial layers, figure source data, validation summaries, metadata, review screens and source manifests.

Restricted raw commercial POI archives, raw web captures, raw downloaded OSM packages and unlicensed full-text source captures are not redistributed. Their source classes, access routes, processing roles and derived outputs are documented in metadata and source-ledger tables. Public OSM-derived context should be re-acquired from the cited OSM or Geofabrik route under the original licence terms when a full raw-data rebuild is required.

Key orientation files:

- `DATA_AND_CODE_AVAILABILITY_NOTE.md`: formal data and code availability statement.
- `DATA_CODE_FIGURE_TABLE_CROSSWALK.md`: manuscript-element crosswalk linking figures, tables and methods to supporting data and code.
- `data_code_figure_table_crosswalk.csv`: editable version of the same crosswalk.
- `metadata/`: source manifests, variable dictionaries, quality gates and processing boundary notes.
- `figure_source_data/`: panel-level source tables for submitted figures.
